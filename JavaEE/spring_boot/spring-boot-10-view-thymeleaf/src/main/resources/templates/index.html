th:text="${#strings.isEmpty(msg)}" 判断某个字符串是否为空

1.Thymeleaf的内置对象，必须使用#

2.Thymeleaf大多数的内置对象都是以s结尾，比如#strings，#numbers，#dates

校验字符是否存在在字符串中 th:text="${#strings.contains(msg,'T')}

字符串中的首字母 th:text="${#strings.startsWith(msg,'T')}

字符串的尾字母 th:text="${#strings.endsWith(msg,'T')}


字符串长度 th:text="${#strings.startsWith(msg,'T')}

字符位置 th:text="${#strings.indexOf(msg,'T')}

字符subString th:text="${#strings.substring()}"






upper lower case th:text="${#strings.toUpperCase(msg)}
